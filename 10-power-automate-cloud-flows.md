# Power Automate (Cloud) Flows

## Flow Name (Title Case with pipes):

```
VCOM OAT | <Domain> | <Verb Noun> | <System?> | <ENV>
# Example: VCOM OAT | VDATA | Import Users | VELENTRA | DEV
```

Child/helper flows: append | Helper or | Child
Scheduled: append | 05:15 CT if you need a visible schedule cue
