# Webflow Naming for Power Automate Cloud Flows

```
Generate a Power Automate Cloud Flow name using VCOM OAT naming standards.

Flow Display Name Format:
"VCOM OAT | <Domain> | <Verb Noun Action> | <System?> | <ENV>"

Rules:
• Title Case  
• Use verbs like: Import, Export, Update, Sync, Validate, Reconcile  
• System is optional—include if the Flow connects to a specific system  
• ENV is always required (DEV, TEST, UAT, PROD)

Given:
• Domain: <insert domain>
• System: <insert system or NONE>
• Action: <insert action, e.g., Import Users>
• Environment: <insert env>

Produce:
1. Flow Display Name
```
